## Browser vs Node

| Browser          | Node.js          |
| ---------------- | ---------------- |
| DOM              | No DOM           |
| WIndow           | No Window        |
| Interactive Apps | Server Side Apps |
| No File system   | File system      |
| Fragmentation    | Version          |
| ES6 Modules      | CommonJS         |
